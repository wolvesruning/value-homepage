# Value 动态个人主页

这是一个可以直接部署到 GitHub Pages 的静态个人网页，内容已根据个人简历整理为项目/产品运营方向，页面里包含动态交互效果：

- Canvas 粒子背景
- 可拖拽互动核心
- 鼠标光晕跟随
- 滚动显现动画
- 数字滚动
- 技能卡片翻转
- 项目卡片 3D 倾斜
- 明暗主题切换

## 部署到 GitHub Pages

1. 把 `index.html`、`styles.css`、`script.js`、`README.md` 放到 GitHub 仓库根目录。
2. 进入仓库 `Settings` -> `Pages`。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。
5. 等待页面生成。

## 可继续补充的内容

- 真实头像或个人照片
- GitHub、LinkedIn、作品集链接
- 微信二维码或更多联系方式
- 更详细的项目案例页面
